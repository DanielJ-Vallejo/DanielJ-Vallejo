# Data / Datos

**EN** — Central data area shared by all projects. `raw/` holds immutable source data,
`processed/` holds derived artifacts. Datasets that cannot be redistributed are
git-ignored; each subfolder documents how to obtain them.

**ES** — Área central de datos compartida por todos los proyectos. `raw/` guarda los
datos fuente inmutables, `processed/` los artefactos derivados. Los datasets que no se
pueden redistribuir están en `.gitignore`; cada subcarpeta documenta cómo obtenerlos.

| Dataset | Source / Fuente | Redistributable |
|---|---|---|
| `raw/titanic/titanic.csv` | Public domain (Kaggle / Data Science Dojo mirror) | ✅ committed |
| `raw/sentiment-analysis/*.xlsx` | MeIA 2025 challenge organizers | ❌ git-ignored |
| `raw/fraud-detection/` | Generated synthetically by `proyecto-01` | ♻️ regenerate with `run_pipeline.py` |
| `raw/simulaciones/` | Generated by the simulations themselves | ♻️ regenerate with `run_all.py` |
