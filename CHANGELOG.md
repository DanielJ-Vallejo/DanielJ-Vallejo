## [1.0.0] — 2026-06-17

### Added
- 12 proyectos de ciencia de datos autocontenidos: detección de fraude, análisis
  de sentimientos (NLP), Titanic, simulaciones de física estadística, visión por
  computadora (LSM + fundamentos + tracking), automatización para negocios
  (bot de Telegram, extractor CFDI, monitor de precios) y analítica SQL
  (demanda del Mundial, predicción de churn).
- Pipeline de CI: tests (matriz por proyecto), linting con ruff y escaneo de
  seguridad (bandit + pip-audit).
- Sitio web de portafolio en GitHub Pages.
- Documentación: arquitectura, guía de contribución y glosario.
